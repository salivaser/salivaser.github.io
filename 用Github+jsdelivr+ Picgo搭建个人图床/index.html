# 配置 PicGo 参数

  • 仓库名：用户名 / 仓库名
  • 分支名： master
  • Token： 粘贴上面复制的 Token
  • 存储路径：自定义
  • 自定义域名： https://cdn.jsdelivr.net/gh/ 用户名 / 仓库名

Github 图床配置完后点击确定再设置为自定义即可，在上传区就可以把图片上传到自己的图床了，上传后可以在相册复制对应格式的图片链接，非常方便哦。
